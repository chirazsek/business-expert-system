# business-expert-system
Expert system for urban business analysis using Prolog + Web interface

## Description
A decision-support system that evaluates business viability using Prolog rules and a modern web interface.

## Features
- 20+ evaluation factors
- Risk analysis
- Success probability scoring
- Smart recommendations

## Technologies
- SWI-Prolog
- HTML / CSS / JavaScript

## How to run
1. Install SWI-Prolog
2. Run:
   swipl server.pl
3. Open:
   http://localhost:8080
